# Code Of Conduct

## Standard

Be respectful, constructive, and specific. Assume good intent, but keep review evidence-based and direct.

## Unacceptable Behavior

- harassment, threats, or personal attacks
- publishing private data, secrets, private snippets, logs, screenshots, or recordings
- pressuring maintainers to bypass review, security checks, or licensing decisions
- knowingly submitting misleading test evidence or generated content as real operator evidence

## Enforcement

The maintainer is the final decision-maker for moderation, pull request acceptance, and repository access. Contributions that violate this code may be edited, closed, or removed.
